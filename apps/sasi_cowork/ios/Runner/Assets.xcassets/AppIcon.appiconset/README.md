# App Icon Set

Icons in this folder are the application icon set (target/bullseye design on blue background).

- **iPhone / iPad**: All required sizes are filled from the provided icon set.
- **App Store (1024×1024)**: Currently a placeholder (scaled from 83.5@2x). For App Store submission, replace `Icon-App-1024x1024@1x.png` with a proper 1024×1024 px PNG.
